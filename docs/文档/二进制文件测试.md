# Binary File Unicode Test

## 测试二进制文件

This document describes testing binary files with Unicode metadata.

### 二进制文件类型
- Python脚本 (.py)
- 二进制数据 (.bin)
- 配置文件 (.yaml)

### 测试内容
- Unicode元数据处理
- 二进制数据生成
- 特殊字符支持

### 使用方法
```bash
# Generate binary file with Unicode
python src/测试文件.py

# Check binary file
file src/测试文件.py
```
